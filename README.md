# Financial Behavior & Expense Analysis 

**📊 Project Overview**

This project is an interactive Power BI dashboard designed to analyze personal financial data, including income, expenses, and savings trends over time.

**🎯 Objective**
- Track total income, expenses, and net savings
- Analyze spending patterns across categories
- Identify savings trends and financial behavior
- Enable data-driven financial decision making

**💻⚙️ Tools & Technologies Used**
- Power BI
- DAX
- Power Query
- Excel
- Data Cleaning
- Data Preparation
- Data Visualization

**✨ Key Features**
- KPI cards: Total Income, Spending Ratio, Savings Rate, Net Savings
-  Monthly trend analysis (Income vs Expense vs Savings)
-  Category-wise expense distribution
-  Savings trend visualization
-  Interactive filters (Month & Year slicers)

**🧹 Data Cleaning & Transformation**
- Removed null and invalid rows
- Handled inconsistent date formats
- Created custom columns
- Built data pipeline using Power Query

**📌 Key Insights**
- Majority of spending is in Home and Grocery categories
- Savings fluctuate significantly across months
- Certain months show high expense spikes impacting savings
- Financial discipline improves over time with controlled spending
